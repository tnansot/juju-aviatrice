# Prérequis académiques, nationalité, anglais

> **Dernière mise à jour : avril 2026**

Cette fiche synthétise les **prérequis transverses non médicaux** communs aux différentes formations de pilote de ligne pertinentes pour Juju. Pour le détail, se reporter à chaque fiche formation.

## Âge

| Formation | Âge minimum | Âge maximum |
|---|---|---|
| ENAC EPL/S | 16 ans | 23 ans (au 1er janvier de l'année du concours) |
| ENAC EPL/U | 17 ans | 28 ans |
| ENAC EPL/P | 18 ans | 30 ans |
| Air France Cadets | ~18 ans | ~27 ans (non formellement bloquant) |
| Lufthansa EFA | 17 ans | pas de limite haute stricte |
| KLM Flight Academy | 18 ans | — |
| BA Speedbird | 18 ans | — |
| easyJet / Ryanair / Wizz | 18 ans | — |
| Iberia Cadets | 18 ans | — |

**Règle générale** : tant que Juju est en cours de cursus scolaire, elle est dans la fenêtre d'éligibilité. L'âge devient contraignant principalement pour l'ENAC EPL/S (plafond à 23 ans), ce qui laisse une marge confortable pour un bac+1 à bac+3 de préparation.

## Nationalité / droit de travailler

| Formation | Exigence |
|---|---|
| ENAC EPL | **UE / EEE** |
| Air France Cadets | **EEE ou Suisse**, + maîtrise du français |
| Lufthansa EFA | **Suisse / UE / EFTA**, passeport non restrictif |
| KLM Flight Academy | Droit de travail UE / EEE |
| BA Speedbird | **Droit de travail UK** (en pratique, résidence UK) |
| easyJet Generation | UK / EEA / UE / Suisse |
| Ryanair / Wizz / Iberia / Brussels | UE / EEE |

**Juju (française)** est éligible nationalité-wise à **toutes les voies européennes continentales**. La seule voie effectivement bloquée en pratique est **BA Speedbird** (réservée aux résidents UK).

## Diplôme / niveau scolaire

| Formation | Diplôme minimum exigé |
|---|---|
| ENAC EPL/S | **Bac+1 scientifique (60 ECTS CPGE ou licence)** |
| ENAC EPL/U | Bac+3 scientifique + ATPL théorique + LAPL/SPL |
| ENAC EPL/P | Bac + CPL + ATPL théorique |
| Air France Cadets | **Bac** (ou 2e année CPGE, sauf Littérature) |
| Lufthansa EFA | Secondaire, anglais ≥ B |
| KLM Flight Academy | Secondaire scientifique |
| BA Speedbird | Secondaire avec résultats en maths, anglais, sciences |
| easyJet Generation | Secondaire (maths + anglais + 1 sciences + 2 autres) |
| Ryanair / Wizz / Brussels | Secondaire scientifique |
| Iberia Cadets | Secondaire, vocation + aptitudes |

**Pour Juju (1ère générale spés scientifiques)** :
- Le **bac scientifique** débloque les programmes cadets sponsorisés (Air France Cadets, Lufthansa, KLM, easyJet…).
- L'**ENAC EPL/S** nécessite en plus **au moins 1 an après le bac**, idéalement en **CPGE PCSI/MPSI** (ou licence scientifique solide avec validation de 60 ECTS).

## Niveau d'anglais

### Exigences formelles

| Formation | Niveau d'anglais attendu |
|---|---|
| ENAC EPL/S écrit | Compréhension écrite générale + vocabulaire aéro (QCM 2h, note éliminatoire 8/20) |
| ENAC EPL/S oral | 15 min, note éliminatoire 10/20 — niveau **B2/C1** |
| Air France Cadets | ~850 TOEFL iBT (L/R) + 170 (Speaking) — équivalent **C1** |
| Lufthansa EFA | **Fluent** (langue d'enseignement) — au minimum note "B" en anglais au certificat scolaire |
| KLM / BA / easyJet / Ryanair / Wizz / Iberia | **Fluent** (langue d'enseignement et d'exercice) |

### Référence — Niveau ICAO

Au-delà de la formation, **tout pilote de ligne exerce en anglais** et doit obtenir/maintenir le niveau **ICAO 4 au minimum** (niveau 6 = expert, sans expiration). L'ICAO évalue 6 dimensions : prononciation, structure, vocabulaire, fluidité, compréhension, interactions.

### Objectif pratique pour Juju

- **Viser C1 avant la fin du lycée** : permet de candidater à toutes les voies sans blocage linguistique.
- **Passer un test de positionnement** (TOEFL, IELTS, Linguaskill) pour objectiver son niveau.
- **S'exposer à l'anglais aéronautique** : vidéos pilotes (Captain Joe, 74 Gear…), ATC (LiveATC), phraseology IFR.

## Casier judiciaire

- **Casier vierge** exigé partout (B2).
- Pour Air France et certaines compagnies : enquête de sécurité complémentaire possible.

## Synthèse pour Juju

Juju, **française, en 1ère générale scientifique**, est en bonne position pour toutes les voies européennes continentales. À travailler en priorité pour maximiser son éligibilité :

1. **Classe 1 médicale** à valider tôt.
2. **Anglais niveau C1** à viser avant le bac (fin de terminale).
3. **Choix post-bac** : CPGE scientifique (idéal pour ENAC EPL/S) OU bac → Cadets Air France directement.
4. **Mathématiques et physique solides** : indispensables pour toutes les voies, niveau prépa pour l'ENAC.
